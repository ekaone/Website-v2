Just use simple `every()` function to check each element than contains based on your condition

```js
[1, 2, 3, 4, 5].every((el) => el < 10); // true
[1, 2, 3, 4, 5].every((el) => el < 5); // false
[1, 2, 3, 4, 5].every((el) => el > 0); // true
```
