Given an elements array.

```js
const elements = [1, 2, 3];
```

We can use spread-operator to add an item.

```js
[0, ...elements];
// output [0, 1, 2, 3]
```

Or use `splice` method.

```js
elements.splice(3, 0, 4);

console.log(elements);
// output [1, 2, 3, 4]
```
